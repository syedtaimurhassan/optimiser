MPSolver - network design MIP

Problem: decide which shipping lanes to open, then route demand through the opened network at minimum fixed plus variable cost.

Show code 
import { initMPSolver, MPSolver } from 'or-tools-wasm/mp-solver';

await initMPSolver();
const solver = MPSolver.CreateSolver('SCIP');
solver.SetNumThreads(4);

const open = arcs.map((arc) => solver.BoolVar(`open_${arc.id}`));
const flow = arcs.map((arc) =>
  demands.map((demand) => solver.NumVar(0, solver.infinity(), `flow_${arc.id}_${demand.id}`)));

for (const [arcIndex, arc] of arcs.entries()) {
  const capacity = solver.Constraint(-solver.infinity(), 0);
  capacity.SetCoefficient(open[arcIndex], -arc.capacity);
  for (const demandIndex of demands.keys()) {
    capacity.SetCoefficient(flow[arcIndex][demandIndex], 1);
  }
}

for (const [demandIndex, demand] of demands.entries()) {
  for (const node of nodes) {
    const rhs = node.id === demand.from ? demand.amount : node.id === demand.to ? -demand.amount : 0;
    const conservation = solver.Constraint(rhs, rhs);
    for (const [arcIndex, arc] of arcs.entries()) {
      if (arc.from === node.id) conservation.SetCoefficient(flow[arcIndex][demandIndex], 1);
      if (arc.to === node.id) conservation.SetCoefficient(flow[arcIndex][demandIndex], -1);
    }
  }
}

const objective = solver.Objective();
for (const [arcIndex, arc] of arcs.entries()) {
  objective.SetCoefficient(open[arcIndex], arc.fixedCost);
  for (const demandIndex of demands.keys()) {
    objective.SetCoefficient(flow[arcIndex][demandIndex], arc.unitCost);
  }
}
objective.SetMinimization();
await solver.Solve();
  • This is a mixed-integer linear program: lane-open decisions are binary, shipment flows are continuous.
  • Opening a lane pays a fixed cost even before any shipment moves on it.
  • Each shipment demand must be conserved through the network from source to destination.
  • A lane can carry flow only if it is opened, and its total flow cannot exceed capacity.
  • Hover a demand or an opened lane after solving to see the related routes.
